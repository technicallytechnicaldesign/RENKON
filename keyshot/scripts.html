Creo to KeyShot automation in the lux / luxmath API: batch views, turntables, and reveal animations. Headless where noted.

Inventory

Conventions

  • Filename: {PREFIX}_{AREA}_{NAME}_{REV}.py
  • PREFIX: 0_ pre-check · 1_ pre-render/helper · 2a_ batch render · 2b_ animation · 3_ post-process · 4_ post-check
  • AREA: 3-letter code: BAT, ANI, HLP, PRC, VAL, CHK…
  • REV: XX01: letter steps for a breaking change, number steps for a routine one
  • Python 3.10+, lux / luxmath modules
  • Headless CLI runs anything marked HEADLESS COMPLIANT; still needs a machine with a licensed KeyShot install
  • Studios (camera + environment + image style) preferred over raw Cameras
  • Binary scene files (.bip, .ksp…) tracked via Git LFS
