Label Generator

Procedural placeable label decals — localized image maps in the KeyShot Label sense. Pick a generator, dial the parameters, and export the opacity / bump / spec (and optional normal) channel set ready to drop into a KeyShot label. Pure browser, seeded & reproducible, no rendering.

Channels: opacity is the real RGBA cutout (alpha = where the label shows) rendered over the material; bump is the height field with mid-grey 128 = flat (Bump Depth scales contrast around 128); spec drives roughness (white = shinier, black = matte); normal is derived from bump via a Sobel filter. Exports name as <basename>_opacity.png / _bump.png / _spec.png / _normal.png — drop the set into the KeyShot label template's channel slots.
